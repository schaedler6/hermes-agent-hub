# 🗺️ Roadmap de Evolução — Hermes Agent Hub

Este documento descreve os marcos planejados para o desenvolvimento futuro do Hermes Hub.

---

## 🎯 Próximas Versões Planejadas

### v0.4.0 — Próxima Versão Menor (Planejado)
*   **Editor Visual de Configuração:** Inclusão de aba interativa no Dashboard para alteração direta das configurações do `config.local.json`.
*   **Suporte Multiplataforma Completo:** Homologação e validação da suíte de testes em ambientes macOS e Linux (através de comandos nativos PowerShell Core).
*   **Melhoria do Validador de Skills:** Suporte a arquivos de metadados expandidos e importações de schemas adicionais.

### v0.5.0 — Indexação e RAG Local (Planejado)
*   **Integração com Obsidian:** Scanner nativo local para ler e mapear relacionamentos entre arquivos do Obsidian e Agent Skills.
*   **Busca Semântica Local:** Mecanismo básico off-line para buscar e indexar skills utilizando representações locais simples.

### v1.0.0 — Estabilização e Comunidade (Longo Prazo)
*   **Marketplace Local de Plugins:** Catálogo interativo apresentando repositórios homologados de plugins de código aberto da comunidade.
*   **Suporte a MCP Estendido:** Painel interativo para registrar e depurar novos servidores MCP de forma totalmente integrada.
