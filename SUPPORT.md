# ℹ️ Canal de Suporte e Ajuda (Support Guidelines)

Se você tiver dúvidas, sugestões ou encontrar algum problema ao rodar o **Hermes Agent Hub**, utilize as seguintes diretrizes para obter suporte:

---

## 1. Pesquisa e Documentação
Antes de abrir um chamado, consulte a documentação local incluída na pasta `docs/`:
*   [docs/INSTALLATION.md](docs/INSTALLATION.md) (Requisitos de sistema e configuração inicial)
*   [docs/QUICK_START.md](docs/QUICK_START.md) (Início rápido e parametrização)
*   [docs/TROUBLESHOOTING.md](docs/TROUBLESHOOTING.md) (Resolução rápida de erros de integridade e trust)

---

## 2. Abrindo Issues no GitHub
Se o seu incidente persistir após consultar o guia de Troubleshooting:
1.  Verifique a seção de Issues do repositório público para atestar se o problema já não foi relatado por outro usuário.
2.  Caso seja um novo problema, crie uma Issue detalhando:
    *   Sua versão do Windows e do PowerShell (`pwsh --version`).
    *   O log de execução do terminal (copie a saída do PowerShell).
    *   Passo a passo detalhado de como reproduzir o comportamento anômalo.
